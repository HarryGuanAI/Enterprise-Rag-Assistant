# 云舟科技 IT 服务指南

## 账号开通

新员工入职前，HR 会向 IT 服务台提交账号开通工单。IT 将在入职当天 12:00 前完成企业邮箱、企业微信、VPN、代码仓库和知识库账号开通。账号初始密码不得通过公开群聊发送。

## 密码重置

员工忘记密码时，应通过 IT 服务台提交“账号与权限”类工单。IT 在核验员工身份后重置密码，并要求员工首次登录后立即修改。任何人不得代替他人索要或保存密码。

## VPN 使用

远程访问内部系统必须使用公司 VPN。员工需要在受信任设备上安装 VPN 客户端，并开启多因素认证。禁止在网吧、共享电脑或未知安全状态的设备上连接公司 VPN。

## 设备报修

电脑、显示器、键盘、门禁卡等办公设备出现故障时，应提交“设备报修”工单。工单需要填写设备编号、故障现象、影响范围和联系方式。影响研发发布或客户交付的问题可标记为高优先级。

## 软件安装

员工安装开发工具、设计软件或数据分析工具时，应优先使用公司软件中心。需要采购商业授权的软件，应由直属主管和 IT 共同审批，禁止使用盗版软件或来源不明的破解工具。
