# 云舟科技数据安全规范

## 数据分级

公司数据分为公开、内部、敏感和受限四级。公开数据可以对外发布；内部数据仅限员工访问；敏感数据包括客户合同、财务数据、员工信息和业务指标；受限数据包括密钥、生产数据库导出和未脱敏客户数据。

## 客户数据处理

处理客户数据时，应遵循最小权限原则。员工只能访问完成工作所必需的数据，不得将客户数据下载到个人网盘、私人电脑或未经批准的外部平台。

## 脱敏要求

在测试、演示、培训或问题排查场景中使用客户数据时，必须进行脱敏。手机号、邮箱、身份证号、访问令牌、合同金额和客户名称应根据场景进行遮蔽或替换。

## API Key 管理

API Key、数据库密码、JWT 密钥和云服务 AccessKey 不得写入代码仓库、README、截图、群聊或工单评论。密钥应存放在环境变量或密钥管理服务中。疑似泄露时应立即废弃旧密钥并重新生成。

## 外部共享

向外部合作方共享数据前，需要确认共享目的、数据范围、接收方、有效期和安全责任。敏感数据和受限数据对外共享必须经过部门负责人和安全负责人审批。
